Frequently Asked Questions

During the courses on web development you will come across with some key terms essential for your development journey. From those some of the important aspects that are frequently asked by the learners are provided.

The main difference between CSS Grid Layout and CSS Flexbox layout is that flexbox is one-dimentional layout, either row or column and grid is a two-dimentional layout, row and column at the same time. Flexbox is less customable and can perform simple tasks. On the otherhand, Grid is highly customable and can perform complex tasks.

Bootstrap comes with a collection of pre-styled responsive, mobile-first components that possess a definite UI kit. On the otherhand, Tailwind CSS uses a set of utility classes to create a neat UI with more flexibility and uniqueness. Sites created using Bootstrap follow the generic pattern that makes them look identical and bootstrap is less customable than Tailwind CSS.

The CSS box model is essentially a box that wraps around every HTML element. It consists of: margins, borders, padding, and the actual contents height and width.

The HTML semantic tags refers to the tags that provide meaning to an HTML page rather than just presentation. It makes HTML more comprehensible by better defining the different sections and layout of web pages.
